# Agent Memory

## 项目概述

**项目名称**：智能旅游规划 Agent（Trip Planner）

**项目目标**：服务懒人规划用户，初期专注中国境内城市，通过 AI 自动生成旅游行程，节省用户制定计划的时间。

## 用户需求

| 维度 | 选择 |
|------|------|
| 目标用户 | 懒人规划（全自动生成行程） |
| 交互形式 | 表单向导式（填写目的地/天数/偏好，生成行程卡片） |
| 技术栈 | Python/FastAPI 后端 + Vue 3 Web 前端 |

## 数据源

- **高德地图 API**：景点位置、路线规划、地理编码
- **携程 API**：酒店、景点门票
- **LLM**：Claude（用于行程生成）
